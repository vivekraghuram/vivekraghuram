Background

Mesh manipulation is extremely important because it is one of the main ways that 3D objects are represented in a computer. Mesh simplification is a useful technique for compressing the size of a given mesh. This is useful for keeping meshes a reasonable size for computers to handle. Regularization through remeshing creates a mesh with roughly consistent voronoi areas for the vertices. The resulting structure is useful for performing certain computations that would be difficult to solve otherwise. In paper 2, remeshing is used when manipulating regions of a mesh in order to perform the manipulation on the simpler remeshing and the mapping the details onto the result.
